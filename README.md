# 📺 YouTube Clone UI

A front-end clone of YouTube's homepage UI, built with pure HTML and CSS — no frameworks, no JavaScript libraries.

---

## 🖥️ Demo

http://127.0.0.1:5500/youtube.html

---

## ✨ Features

- Fixed Header — YouTube-style navbar with hamburger menu, logo, search bar with search & voice search buttons, upload, apps, notifications (with count badge), and user profile icon
- Collapsible Sidebar — Fixed left sidebar with navigation links: Home, Explore, Subscriptions, Originals, YouTube Music, and Library
- Responsive Video Grid — Displays 12 video cards that adapt from 2 → 3 → 4 columns based on screen size
- Video Cards — Each card includes a thumbnail, video duration badge, channel profile picture, video title, author name, and view/time stats
- Hover Tooltips — Tooltips appear on hover for Search, Voice Search, Create, YouTube Apps, and Notifications buttons
- Google Fonts — Uses Roboto font (400 & 500 weights) matching YouTube's typography
- Responsive Design — Adapts layout for mobile, tablet, and desktop screens

---

## 🛠️ Tech Stack

- HTML5 — Semantic structure and layout
- CSS3 — Flexbox, CSS Grid, media queries, positioning, transitions
